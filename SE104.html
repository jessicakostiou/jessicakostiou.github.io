Young and Old Robots

This project was a precursor to my honors thesis. This course explored introductory level programming and applied it to Scribbler S2 Robots. The goal of this final project was to design two different personas to the robot, one young and one old. This was accomplished by programming the robot with varying assortments of movements, light signals, and sounds.

Prior to programming, requirements, use cases and test cases were completed to serve as guidelines for how the project would be created. This documentation outlined the way the robot's software would be designed, and was followed through the development process.
