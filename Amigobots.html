Amigobot Programming

Amigobots are wireless robots that contain wireless routers. These small embeded systems are able to comunicate with C++ programs using a client-server architecture. Visual studio was used as a programming enviornment to write programs to manipulate and navigate the robots. The final project for this course was programming multiple mail slots to allow four robots to dance for one to two minutes to a song. The video below demonstrates the robots them dancing to to “Single Ladies” by Beyoncè.
