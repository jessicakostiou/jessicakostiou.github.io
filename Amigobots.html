Amigobot Programming

Amigobots are wireless robots that contain wireless routers. These small embedded systems are able to communicate with C++ programs using a client-server architecture. Visual studio was used as a programming environment to write programs to manipulate and navigate the robots. The final project for this course was programming multiple mail slots to allow four robots to dance for one to two minutes to a song.
